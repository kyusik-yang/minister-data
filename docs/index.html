National Assembly Q&A Dyads — Primary Research Use Case

What is a Q-A dyad?

South Korea's LOSI system archives full-text transcripts of every committee hearing and national audit session since 2004. Each legislator–minister exchange can be parsed into a question-answer dyad — one row per spoken turn.

This minister panel provides the dual-office key to join onto any LOSI-derived dyad dataset, enabling analysis of whether co-partisan ties between legislators and ministers distort legislative oversight.

HEARING 인사청문회 AUDIT 국정감사 274,763 exchanges total 17th–21st Assembly

# Three-way merge: dyads + minister panel + MP metadata dyads = pd.read_csv("your_losi_dyads.csv") panel = pd.read_csv("minister_panel_comprehensive.csv") meta = pd.read_csv("losi_mp_metadata.csv") # Step 1: attach dual-office status dyads = dyads.merge(panel[["name", "admin", "ministry", "dual_office"]], left_on=["minister", "admin", "ministry"], right_on=["name", "admin", "ministry"], how="left") # Step 2: attach questioner party dyads = dyads.merge(meta[["q_speaker", "assembly", "q_party"]], on=["q_speaker", "assembly"], how="left") # Step 3: analyze oversight patterns dyads.groupby(["dual_office", "q_ruling"])["q_word_count"].mean()
